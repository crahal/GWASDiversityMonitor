About

This interactive dashboard monitors the diversity of participants across all published Genome Wide Association Studies, a commonly utilised type of observational study which scans the genome. The objective is to identify statistical associations between a set of genetic variants across different individuals (Single Nucleotide Polymorphisms, or 'SNPs') and specific traits of interest. It represents an extension of our earlier project, 'The Scientometrics of Genome Wide Association Studies' published in Communications Biology in January 2019. As in that work, we leverage the magnificent dataset provided by the NHGRI-EBI Catalog to which me make full attribution (and their licensing information can be found here). The dashboard itself is written in Bokeh 1.2.0, an interactive visualization library in Python which targets modern web browsers for presentation. The source code is hosted on GitHub, and we are actively encouraging community based suggestions and contributions. The backend code checks daily for updates to the GWAS Catalog, and when there is an update, it writes to a log file and refreshes the dataset which powers the dashboard. The dashboard itself is hosted on Heroku (a PaaS). The dashboard is under constant development and review, but is currently comprised of five widgets: 'Evaluation Metric', 'Broader Ancestry', 'Parent Term', 'Research Stage', 'Year'. Due to the relatively small size of the dataset, not all widgets apply to each figure, and we describe our design choics below:

  • Figure 1: Bubble Plot. This provides a granular overview of all GWAS in the Catalog. It maps onto EFO Parent terms, and splits on commas. If a GWAS studies multiple Parent terms, both are included (i.e., we split on commas in a string such as 'Body Measurement, Cancer' and include both). Applicable widgets are: 'Broader Term', 'Parent Term' and 'Research Stage'.

  • Figure 2: Horizontal Bar Plot. This figure extracts ancestral terms from the free text description of a study provided by the Catalog using a regex function as per our scientometric review. Applicable widgets are: 'Parent Term' and 'Research Stage'.

  • Figure 3: Dohnut Chart. This figure displays the percent of which 'Broader Ancestry' is used to study which 'Parent Term' across the two definitions of the 'Evaluation Metric'.

  • Figure 4: Time Series. This figure displays changes in how 'Broader Ancestry' has changed over time across the two 'Evaluation Metric' widgets. Note: due to the fact that that lesser studied ancestries are often not used for to study specific parent terms, we do not utilize the 'Parent Term' widget here.

  • Figure 5: Choropleth Map. Due to the sparisty of countries recruited from across all EFO Parent terms, this figure online calls back to two widgets: the 'Evalaution Metric' and the 'Year' slider which determines year of recruitment. This is also partially due to the lack of clarity in the free-text of academic articles which are catalogued by the NHGRI-EBI.
